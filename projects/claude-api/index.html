Puter.js + Claude

Started over. Chat-first. Smaller surface. Better operator flow.

The old Claude page was overbuilt. This version is a fresh implementation centered on the only things that matter: model selection, system context, attachment ingestion, deterministic request preview, and a usable transcript.

  • Live browser execution against puter.ai.chat() with streaming or buffered responses.
  • Attachment ingestion for text, markdown, JSON, JavaScript, TypeScript, HTML, CSS, Solidity, and log files.
  • Inspectable prompt assembly so the actual request envelope is visible before execution.
